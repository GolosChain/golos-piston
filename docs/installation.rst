Installation
************

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install piston-lib piston-cli

Manual installation:

::

    $ git clone https://github.com/xeroc/piston-lib
    $ cd piston-lib
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
